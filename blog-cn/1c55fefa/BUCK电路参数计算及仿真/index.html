BUCK电路性能指标

1、输入电压：标准直流电压48V;

2、输出电压：直流电压24V，5A;

3、输出电压纹波：100mV;

4、电流纹波：0.25A;

5、开关频率：fs = 250kHz

需要计算的参数

1、占空比D1;

2、电感L

3、电容C

4、开关元件Q的选取

BUCK电路

参数计算使用到的公式

1、法拉第定律：$\cfrac{V_{L}}{L}=\cfrac{\Delta{I_{L}}}{\Delta{t}}$

2、伏秒值相等的原则：$V_{on}T_{on}+V_{off}T_{off} = 0$

3、电容的定义：$C = \cfrac{\Delta{Q}}{\Delta{U_c}}$

4、对于上图（B）,应用基尔霍夫电压定律可得：$U_{L} = U_{d}-U_{o}$

5、对于上图（C）,应用基尔霍夫电压定律可得：$U_{L} = -U_{o}$

6、对电感L应用“伏秒值相等的原则”：$(U_d - U_o)D_1T_s - U_o(1-D_1)T_s = 0$

7、由上面6可得：$D_1 = \cfrac{U_o}{U_d}$

8、由法拉第定律$\cfrac{V_{L}}{L}=\cfrac{\Delta{I_{L}}}{\Delta{t}}$即$\cfrac{V_{L}}{L}=\cfrac{\Delta{I_{L}}}{D_1T_s}$可得：$L = \cfrac{U_LD_1T_s}{\Delta{I_L}}$

9、由电容C充电的电荷为：$\Delta{Q} = \cfrac{\Delta{I_L}T_s}{8}$可得$C = \cfrac{\Delta{I_L}T_s}{8\Delta{U_c}}$

simulink仿真模型

参数计算及设置

1、DC Voltage Source模块参数设置

输入电压为48V

2、Pulse Generator模块参数设置

由开关频率为250kHz得，Period =$\cfrac{1}{250000}$=0.000004 secs

由$U_o = 24V$和$U_d = 48V$得占空比 $D_1 = \cfrac{24}{48} =50$ %

3、Series RLC Branch (Branch type:R)参数设置

由输出电压24V，电流5A可得，Resistance = $\cfrac{24}{5} = 4.8 \Omega$

4、Series RLC Branch (Branch type:L)参数设置

$U_L = U_o - U_d =48 -24=24V$

$D_1 = 0.5$ （占空比）

$T_s = 0.000004$ （周期 s）

$\Delta{I_L} = 0.25$ （电流纹波 A）

$L = \cfrac{U_LD_1T_s}{\Delta{I_L}} = \cfrac{24×0.5×0.000004}{0.25} = 0.000192$ H

5、Series RLC Branch (Branch type:C)参数设置

$\Delta{I_L} = 0.25$ （电流纹波 A）

$T_s = 0.000004$ （周期 s）

$\Delta{U_c} = 100mV = 0.1V$ （输出纹波电压）

$C = \cfrac{\Delta{I_L}T_s}{8\Delta{U_c}} = \cfrac{0.25×0.000004}{8×0.1} = 0.0000125$ F

6、PowerGUI模块

PowerGUI模块可以选择这些方法来解决电路中的一个：

  • 连续的，其使用从Simulink中的可变步长求解器®
  • 固定时间步长的电气系统离散化解决方案
  • 连续或离散相量解决方案
